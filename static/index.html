tode

the online database of equations.

search quick random example

tode is an open source database to which anybody can contribute. You can search through it using the field above. Pressing "search" will perform a normal search; "quick" will instantly go to the first entry found; "random" will select a random entry; and "example" will fill out the search field with an exemplar query.

To add an equation to the tode database, you'll first need to make an account. Then, click the button at the bottom of this page saying "new entry" (under "general"). You will be prompted to enter the equation along with information such as a description and categories.

tode has a few other services, for example you can download a serial dump of the database. There's also a HTTP API, for which you can generate a key once you have an account. The API supports operations such as querying and selecting a random entry.
